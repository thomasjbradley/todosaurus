Keyboard Shortcuts

Movement

j ↓ Move the focus down
k ↑ Move the focus up
g g ⌘↑ Move focus to the first item
G ⌘↓ Move focus to the last item
⌃d ⇟ Move focus down 5 items
⌃u ⇞ Move focus up 5 items
⌘j Jump to a specific line

Editing

n ⌘N New todo at the top
⇧N ⌘⇧N New todo at the bottom
o ⌘↩ New todo after the focused item
O ⌘⇧↩ New todo before the focused item
x Space Mark/unmark todo as complete
1 2 5 Add priority to todo
0 Remove priority
e ↩ Update an item, selecting all the text
i ⇧I Edit focused the item, cursor at the start
a ⇧A Edit focused the item, cursor at the end
⇧C Edit focused the item, delete all current content
d d ⌘⌫ ⌦ ⌘X Delete todo, move to buffer, copy to clipboard
⇧Y ⌘C Copy todo item to buffer and clipboard
p ⌘V Paste top buffer item below the focused item
⇧P ⌘⇧V Paste top buffer item above the focused item
y p ⌘⇧D Duplicate line, add afterwards
d d k ⇧P ⌘⇧↑ Move item up
d d p ⌘⇧↑ Move item down

Manipulating the view

/ ⌘F Search todo items
⌘L ⌃L Clear search terms
⌘P Filter by project
⌘T Filter by context
⌘1 ⌘2 ⌘9 Display single project
⌥1 ⌥2 ⌥9 Display single context
⇧1 ⇧2 ⇧5 Display single priority
⌘0 ⌥0 ⇧0 Clear tag filter
⌘⇧L ⌃⇧L Clear all filters

Manipulating the file

⌘R Reload the file
⌘⇧S Sort the contents of the file
⌘S Archive all the completed items into “done.txt”
⌘O Switch to another directory
